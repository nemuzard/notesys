
## Overview
This is a full-stack note and problem-set management system originally built for Chinese-speaking students preparing for coding interviews.

- Backend: Java, Spring Boot, MySQL, Redis
- Frontend: React, TypeScript, Vite, Ant Design
- Users can create notes, organize questions into custom problem sets, and manage favorites and messages.

## Features

- User registration, login, and profile management
- Create, edit, and organize notes
- Tag and category management for notes
- Manage problems and custom problem sets 
- Favorites/collections
- Message center 
- Admin dashboard for users, categories, problems, and problem sets

## Tech Stack

- **Backend:** Java, Spring Boot, MySQL, Redis
- **Frontend:** React, TypeScript, Vite, Ant Design
- **Build & Tooling:** Maven, npm
